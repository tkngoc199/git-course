1.
git commit --amend: để chỉnh sửa nội dung commit gần nhất
(sửa commit msg gần nhất, đi vào giao diện của VIM)
git commit --amend -m"<message>": sửa trực tiếp message của commit cuối cùng
(sửa commit msg gần nhất, không đi vào giao diện của VIM, sửa trực tiếp msg)
git restore --staged <file>:là di chuyển file từ staging <file> -> working
git reset HEAD~1: lấy commit số 1 từ repository -> working directory
